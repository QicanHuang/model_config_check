# 功能介绍：
用于检测atp系统中某风电场的‘配置面板’内的配置是否正常，
比如检测‘区域编号’中的容量设置是否正常，‘指定模型’中只当的模型是否正常等等

# 调用方法
```
wind_farm_check(wfid=['320908'])
wind_farm_check(wfid=['320908', '836610'])
```
